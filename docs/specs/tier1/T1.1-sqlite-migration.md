## Database Schema

```sql
-- Source/database.py
CREATE TABLE IF NOT EXISTS users (
    email TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    password_hash TEXT,
    tier TEXT NOT NULL DEFAULT 'free',
    subscription_id TEXT,
    payment_provider TEXT,
    created_at TEXT NOT NULL,
    email_verified INTEGER NOT NULL DEFAULT 0,
    last_login TEXT,
    failed_login_attempts INTEGER NOT NULL DEFAULT 0,
    last_failed_login TEXT,
    oauth_provider TEXT,
    oauth_id TEXT,
    profile_picture TEXT
);

CREATE INDEX IF NOT EXISTS idx_users_oauth ON users(oauth_provider, oauth_id);
```

## Source/database.py Requirements

1. `get_db(db_path="data/mergenix.db") -> sqlite3.Connection`
   - Use WAL mode: `PRAGMA journal_mode=WAL`
   - Enable foreign keys: `PRAGMA foreign_keys=ON`
   - Set busy timeout: `PRAGMA busy_timeout=5000`
   - Use `check_same_thread=False` for Streamlit
   - Create tables if not exist
   - Return connection singleton (use module-level caching)

2. `migrate_json_to_sqlite(json_path="data/users.json", db_path="data/mergenix.db")`
   - Read existing users.json
   - Insert each user into SQLite
   - Only run if users table is empty AND users.json has data
   - Log migration count

3. `init_db(db_path="data/mergenix.db")`
   - Call get_db to create tables
   - Call migrate_json_to_sqlite if needed

## AuthManager Changes

### Constructor

```python
def __init__(self, db_path: str = "data/mergenix.db"):
    self.db_path = db_path
    self._conn = None
    self._init_db()
```

### Replace \_load_users/\_save_users with SQL queries

Every method that currently does:

```python
users = self._load_users()
# modify users[email]
self._save_users(users)
```

Must become:

```python
conn = self._get_conn()
conn.execute("UPDATE users SET ... WHERE email = ?", (...))
conn.commit()
```

### Specific method changes:

- `register_user()` → INSERT INTO users
- `authenticate()` → SELECT + UPDATE in single transaction (no more triple load/save)
- `get_user()` → SELECT WHERE email = ?
- `update_last_login()` → UPDATE SET last_login = ?
- `record_failed_login()` → UPDATE SET failed_login_attempts = failed_login_attempts + 1
- `check_lockout()` → SELECT failed_login_attempts, last_failed_login + conditional UPDATE
- `update_user_tier()` → UPDATE SET tier = ?, subscription_id = ?, payment_provider = ?
- `change_password()` → SELECT password_hash + UPDATE password_hash
- `create_oauth_user()` → INSERT INTO users with OAuth fields
- `link_oauth_account()` → UPDATE SET oauth_provider = ?, oauth_id = ?
- `get_user_by_oauth()` → SELECT WHERE oauth_provider = ? AND oauth_id = ? (indexed!)
- `logout()` → No DB change needed (only clears session state)

### CRITICAL: Preserve public API exactly

All method signatures and return types MUST stay identical:

- `register_user(email, name, password) -> tuple[bool, str]`
- `authenticate(email, password) -> tuple[bool, dict | None]`
- `get_user(email) -> dict | None`
- `get_user_by_id(user_id) -> dict | None`
- `update_last_login(email) -> bool`
- `record_failed_login(email) -> bool`
- `check_lockout(email) -> bool`
- `update_user_tier(email, tier, subscription_id, provider) -> bool`
- `change_password(email, old_password, new_password) -> tuple[bool, str]`
- `create_oauth_user(email, name, provider, oauth_id, profile_picture) -> tuple[bool, str]`
- `link_oauth_account(email, provider, oauth_id, profile_picture) -> bool`
- `get_user_by_oauth(provider, oauth_id) -> dict | None`
- `logout() -> None`
